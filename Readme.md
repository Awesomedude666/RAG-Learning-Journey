# RAG Learning Journey 

This repository contains my step-by-step journey of learning and implementing Retrieval-Augmented Generation (RAG) systems from scratch.

The goal is not just to use existing frameworks, but to understand and build every component involved in a modern RAG pipeline, starting from basic search engines to advanced conversational and metadata-aware RAG systems.

---

## Learning Roadmap

### 01. Build a Tiny Search Engine
- Basic document retrieval
- Keyword matching
- Ranking documents

### 02. Implement Cosine Similarity
- Vector representations
- Similarity calculations
- Nearest neighbor retrieval

### 03. Create Your Own Embeddings Search
- Text embeddings
- Semantic search
- Embedding-based retrieval

### 04. Build a Mini Vector Database
- Store embeddings
- Efficient retrieval
- Index management

### 05. Build RAG Without LangChain
- End-to-end RAG pipeline
- Document ingestion
- Retrieval + Generation
- Understanding what happens under the hood

### 06. Automatic Chunking
- Fixed-size chunking
- Recursive chunking
- Chunk overlap strategies
- Optimizing retrieval quality

### 07. PDF Chatbot (Real-World RAG Project)
- PDF ingestion
- Document chunking
- Semantic search
- Question Answering over PDFs

### 08. Implement FAISS
- FAISS indexing
- Similarity search at scale
- Faster retrieval

### 09. Conversational RAG
- Chat history management
- Context-aware retrieval
- Multi-turn conversations

### 10. Metadata-Aware RAG
- Metadata filtering
- Source-aware retrieval
- Improved relevance
